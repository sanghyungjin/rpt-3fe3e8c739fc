1. 연구 개요

Podarcis muralis (벽도마뱀) 및 근연종 Desmoplakin 단백질에서 **화장품 활성 펩타이드를 AI 기반으로 탐색**하는 연구입니다. v1~v6까지 총 6개 버전의 연구를 진행했으며, 이 문서는 **연구 방법론의 적합성과 향후 방향에 대한 전문가 의견**을 구하기 위해 작성되었습니다.

2. 피드백 요청 항목

2.1 펩타이드 추출 방법론 및 전략

v1~v6에서 사용한 추출/탐색 전략의 **생물학적 타당성과 개선점**에 대한 의견을 구합니다.

# 방법 설명 적용 단계 결과 피드백 요청
1 **천연 추출** (Natural Chopping) Desmoplakin 서열에서 k-mer (4~8AA) 직접 절단 v1, v3 B-grade max (IKDKL 0.87) 천연 유래 claim의 규제/마케팅 타당성?
2 **교차 조합** (Cross-Source) 복수 단백질(10종 DSP) 모티프를 교차 조합 v2-3, v2-4 B-grade 다수 (VPRVR 0.87) 교차 조합 펩타이드의 "천연 유래" 주장 가능 여부?
3 **모티프 점돌연변이** (Motif Mutation) 고활성 모티프의 단일 아미노산 치환으로 최적화 v2-7, v3-2 148 B-grade, CPP 병목 발견 점돌연변이 접근의 활성 유지 가능성? (in silico vs wet lab 차이)
4 **전수 조사** (Exhaustive Search) 4-mer 전체 조합 (20⁴ = 160,000) 스크리닝 v2-8 **A-grade 3개 달성 (ARIK 0.94)** 전수 조사로 발견된 펩타이드의 신뢰도? "natural origin" 아닌 서열의 가치?
5 **Human DSP PRD 집중 마이닝** Human Desmoplakin C-terminal PRD (1,115aa) 세분화 v3-1, v4 941 B-grade, 0 A-grade Human origin이 규제/마케팅에 미치는 영향?
6 **도메인 기반 마이닝** 11개 sub-domain별 conservation + collagen score v4 도메인별 활성 차이 확인 Conservation 기반 펩타이드 선별의 생물학적 근거?
7 **Cross-species Conservation** 4종(Human, Anole, Muralis, Lilfordi) DSP 서열 정렬 v4, v5 Linker B-C 100% 보존 종간 보존 = 기능적 중요성 해석의 타당성?
**Conservation Score 정의**: 4종(Human, *A. carolinensis*, *P. lilfordi*, *P. muralis*)의 Desmoplakin 서열을 다중 정렬(MSA) 후, 도메인 내 각 아미노산 위치에서 **종간 일치율의 평균**을 산출한 값(0~1).

| Score | 의미 | 예시 |
|-------|------|------|
| **1.000** | 4종 모든 위치에서 아미노산 **완전 동일** | Linker B-C (16 aa) — 3억 년+ 종분화에도 불변 |
| **0.807** | 약 81% 위치 동일/유사 | C-terminal PRD (1,115 aa) |
| **0.610** | 약 61%만 보존, 진화적 변동 큼 | Central Rod |

**해석**: 높은 보존도 = 기능적으로 필수적인 서열. 보존된 영역에서 추출된 펩타이드는 종간 기능이 유지될 가능성이 높으며, 생물학적 활성의 진화적 근거를 제공.

**핵심 질문**:

  • 천연 추출(method 1) vs 전수 조사(method 4)의 규제적/과학적 트레이드오프는?
  • 전수 조사에서 나온 ARIK가 Desmoplakin 서열에 존재하지 않는데, "Desmoplakin 유래 연구"로 포지셔닝 가능한지?
  • Human origin(method 5) 펩타이드가 A-grade를 못 찍는 건 구조적 한계인지, 스코어링 한계인지?

2.2 점수 체계 (Scoring System)

4개의 점수 체계가 연구 과정에서 진화했습니다. 각 체계의 **과학적 타당성**에 대한 의견을 구합니다.

2.2.1 Unified Score (v1~v3) — 5 components

Unified = 0.30×Collagen + 0.25×CPP + 0.20×Stability + 0.15×Skin + 0.10×LL37
Component Weight 방법 근거 한계
Collagen Activity 0.30 FusPBESM2 (ProtBERT+ESM2 fusion, fine-tuned) 화장품 핵심 claim 모델이 "collagen fragment"를 예측 (promoter vs fragment 구분 불가)
CPP 0.25 R/K 양전하 AA 비율 기반 휴리스틱 Langel 2019, cationic residue ∝ cell penetration 실제 CPP 활성은 2차 구조 의존적
Stability 0.20 Instability Index < 40 → 1.0 (binary) Guruprasad 1990 4-5AA에서 거의 항상 1.0 → 변별력 없음
Skin Penetration 0.15 MW + GRAVY + charge 복합 Rule of 500 (Bos 2000) MW<500 step function이라 사실상 length proxy
LL37 Similarity 0.10 Sliding window substring match 항균 이중 기능 37AA 대비 4AA match는 biological relevance 낮을 수 있음

2.2.2 Longevity Score (v4) — 7 components

Longevity = 0.25×Col + 0.20×CPP + 0.15×Stab + 0.10×Skin + 0.10×LL37 + 0.10×AA_sim + 0.10×Cons

추가된 2개 component:

Component Weight 방법 의도
Anti-aging Similarity 0.10 GHK/KTTKS 등 reference 서열과 유사도 기존 항노화 펩타이드와의 연관성
Conservation 0.10 4종 cross-species alignment hotspot 진화적으로 보존된 = 기능적으로 중요

**문제**: 가중치 분산으로 이론적 최대 ~0.894 → A-grade(0.90+) 수학적으로 불가능

2.2.3 Viability Score (v5) — 7 components (재설계)

Viability = 0.30×Col + 0.25×CPP + 0.15×Stab + 0.08×Skin + 0.07×LL37 + 0.07×AA_sim + 0.08×Cons

**변경 근거**: Collagen(0.25→0.30) + CPP(0.20→0.25)로 핵심 MOA 가중치 집중

2.2.4 Viability + CPP v2 Ensemble (v6) — 7 components + CPP ML

Viability(v6) = 0.30×Col + 0.25×CPP_ensemble + 0.15×Stab + 0.08×Skin + 0.07×LL37 + 0.07×AA_sim + 0.08×Cons
CPP_ensemble = 0.35 × Rule_v2 + 0.65 × ML_v3

**v6 핵심 변경**: CPP component가 단순 R/K 휴리스틱에서 **Rule v2 + ML v3 ensemble**으로 업그레이드

Component 설명 성능
**Rule v2** 6-component heuristic (R/K differentiation 개선) N/A (rule-based)
**ML v3** FusPBESM2 기반, cpp_dataset_v3_enriched 학습 AUC-ROC 0.9606
**Ensemble** 0.35 × Rule_v2 + 0.65 × ML_v3 Rule + ML 상호보완

**v6에서 해결된 기존 피드백**:

  • CPP를 R/K 비율 휴리스틱으로 평가하는 것의 한계 → **Rule v2 + ML v3 (AUC 0.96) ensemble로 해결**
  • CPP 학습 데이터셋 구축이 필요한지? → **cpp_dataset_v3_enriched 구축 완료 (5,349 sequences, CD-HIT clustered)**

**새로운 피드백 요청** (v6 CPP 변경 관련):

  • **VQLK(Val-Gln-Leu-Lys)**가 새 1위로 부상했는데, 이 서열의 cell penetration potential이 생물학적으로 타당한지? (Lys 1개 + amphipathic 특성)
  • **4-mer peptide의 실제 CPP 활성**: ARIK(4AA)가 in silico에서 높은 cationic ratio를 보이지만, 4개 아미노산으로 실제 세포막 투과가 가능한지? 최소 유효 길이에 대한 문헌 근거는?
  • **IKKV, RGLV 등이 CPP 재평가에서 급락** (ML 모델이 non-CPP로 판단) — 이 서열들이 실제로도 cell penetration이 어려운지, wet lab 검증 없이 판단할 수 있는 기준은?
  • CPP 활성 검증을 위한 **gold standard 실험**: FITC-labeling + confocal? PAMPA assay? 어떤 방법이 가장 적합한지?
  • 5-component vs 7-component 중 어떤 접근이 더 적합한지?
  • 각 component의 가중치 배분이 합리적인지?
  • Stability가 binary(0 or 1)인 것의 문제? 연속값으로 변경해야 하는지?
  • LL37 유사성의 biological relevance: 4AA가 37AA 대비 의미 있는 유사성인지?
  • Anti-aging reference similarity가 scoring에 적합한 component인지?

**Hybrid**: DSP 5-mer + constrained mutation DSP 천연 5-mer에 ±1 mutation ~22,000 수 분

**피드백 요청**:

  • 4-mer에서 A-grade가 나왔지만 5-mer/6-mer 확장이 의미 있는지?
  • 길이 증가 → skin penetration 감소 → 다른 component에서 보상 필요
  • 4AA 최적 vs 5AA 더 다양한 가능성 중 어느 쪽에 투자할지?
  • "기존 연구 피드백 이용한 전수조사" 접근:
  • v1~v4에서 발견한 고활성 모티프(IK, KL, RV, PR 등)를 제약조건으로 사용
  • 예: "반드시 IK 또는 RK 모티프를 포함하는 5-mer"만 전수조사
  • 이런 모티프 기반 constrained search가 과학적으로 합리적인지?
  • Desmoplakin 서열에 존재하지 않는 서열의 가치:
  • ARIK는 DSP에 없지만 A-grade — "천연 유래" 아닌 "AI 설계" 포지셔닝 필요
  • DSP 내 존재하는 서열만 대상으로 하는 전수조사가 더 적합한지?

Appendix: 전문용어 및 약어 정리

약어/용어 정의 비고
**Desmoplakin (DSP)** Desmosome의 핵심 구성 단백질. 중간섬유를 세포막에 연결 본 연구의 주요 단백질 소스
**Desmosome** 세포간 접합 구조. 상피세포를 단단히 연결하여 조직 강도 부여
**Podarcis muralis** 벽도마뱀. 남유럽 원산 파충류, 꼬리 autotomy 후 완전 재생
**PRD** Plakin Repeat Domain — Desmoplakin C-terminal 도메인, 중간섬유 결합 활성 펩타이드 핫스팟
**CPP** Cell-Penetrating Peptide — 세포막 투과 펩타이드. R(Arg), K(Lys) 양전하 잔기 핵심 가중치 0.25
**LL37** Human cathelicidin antimicrobial peptide — 인체 선천면역 항균 펩타이드 (37 AA)
**Collagen** 콜라겐 — 결합조직의 주요 구조 단백질. 피부 탄력과 견고함 가중치 0.30
**FusPBESM2** ProtBERT + ESM-2 fusion classification head. Collagen 및 CPP 활성 예측 자체 개발
**ProtBERT** 단백질 서열 사전학습 언어 모델 (Elnaggar et al., 2021) 1024-dim
**ESM-2** Meta AI 단백질 언어 모델, 6.5억 파라미터 (Lin et al., 2023) 1280-dim
**ESMFold** Meta AI 구조 예측 모델. pLDDT로 신뢰도 표시 v5 structural gate
**AlphaFold2** Google DeepMind 단백질 구조 예측 AI. Multimer 버전은 복합체 예측 가능
**AUC-ROC** Receiver Operating Characteristic Curve 하면적. 1.0=완벽, 0.5=무작위
**AUC-PR** Precision-Recall Curve 하면적. 불균형 데이터에 적합
**pLDDT** predicted Local Distance Difference Test. 구조 예측 신뢰도 (0-100)
**MW** Molecular Weight, 분자량 (Da). <500 Da = 피부 투과 유리
**GRAVY** Grand Average of Hydropathy. 양수=소수성, 음수=친수성
**Instability Index** 단백질 안정성 예측 지표. <40 안정, ≥40 불안정 (Guruprasad, 1990)
**OOD** Out-of-Distribution. 학습 데이터 분포 밖의 샘플
**CD-HIT** 서열 유사도 기반 클러스터링 도구. 중복/유사 서열 제거
**Optuna** Hyperparameter 자동 최적화 프레임워크 (Bayesian optimization)
**ADME** Absorption, Distribution, Metabolism, Excretion — 약물의 체내 동태
**GHK-Cu** Glycyl-Histidyl-Lysine copper complex. 콜라겐 합성 촉진 Positive control
**KTTKS (Matrixyl)** Pentapeptide. TGF-beta 경로로 collagen 간접 촉진 Positive control
**Matrikine** ECM 유래 신호 펩타이드. 세포 활성 조절 생물활성 단편
**Amphipathic** 양친매성. 소수성·친수성 영역 동시 보유. CPP 핵심 특성
**MOA** Mechanism of Action — 약물/펩타이드의 생물학적 작용 기전
**In silico** 컴퓨터 시뮬레이션/예측 기반 분석 본 연구 방법론
**In vitro** 시험관 내 실험 (세포·조직 수준) 향후 검증
**FITC** Fluorescein Isothiocyanate. 형광 표지, CPP 투과 확인
**PAMPA** Parallel Artificial Membrane Permeability Assay. 인공 막 투과 측정
**ELISA** Enzyme-Linked Immunosorbent Assay. 항체 기반 단백질 정량
**MLCPP 2.0** Machine Learning-based prediction of CPP — CPP 예측 벤치마크 DB
**k-mer** 길이 k의 연속 부분 서열. 예: 4-mer = ARIK
**Rule of 500** Bos & Meinardi (2000) — MW 500 Da 미만 물질이 피부 각질층 통과 가능
