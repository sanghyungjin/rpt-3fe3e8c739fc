기본 정보 (컬럼 1-5)

# 컬럼명 타입 설명 값 예시
1 Rank int Score 기준 순위. 각 CSV 내에서 1부터 100까지 순차 1, 2, 3, ...
2 Sequence str 아미노산 서열. Single-letter amino acid code (대문자) RPYGSHREKRGHKG
3 Length int 아미노산 서열 길이 14
4 Scoring_Scheme str 사용된 스코어링 체계. 모든 행이 DEJ_V8 DEJ_V8
5 Score float DEJ_V8 종합 점수. 9개 component의 가중합 (0~0.65 범위) 0.6412
Score 계산: Score = Σ(component_i × weight_i). 9개 component의 가중합이며, DEJ_V8 체계에서 이론적 최대값은 약 0.65입니다.

Grade 2-Step System (컬럼 6-8)

# 컬럼명 타입 설명 가능한 값
6 Raw_Grade str Score 기반 1차 등급 A (≥0.80), B (≥0.50), C (≥0.42), F (<0.42)
7 Final_Grade str Cosmetic viability 강등 후 최종 등급 A, B, C, F
8 Viability str Cosmetic viability 판정 결과 PASS (Raw=Final), DEMOTED (Raw≠Final)
2단계 Grade 체계: Raw_Grade는 순수 score threshold 기반. Final_Grade는 cosmetic viability 평가 (양전하 과다, 단일 잔기 지배, protease 취약성) 후 강등될 수 있음. 강등된 경우 Viability = DEMOTED.

FTO (Freedom to Operate) 분석 (컬럼 9-14)

# 컬럼명 타입 설명 가능한 값
9 FTO_Status str 특허/상용 서열 충돌 상태 BLOCKED (사용 불가), CAUTION (주의, 알려진 motif 포함), CLEAR (충돌 없음)
10 FTO_Reason str FTO 판정 사유 "전체 서열(6aa) 100% 특허 매칭", "부분 매칭만(8aa) — 전체 서열 novel" 등
11 PCPC_Status str PCPC/INCI 화장품 원료 등재 상태 REGISTERED (이미 INCI 등재), NOVEL_WITH_KNOWN_MOTIF (알려진 motif 포함하나 전체 서열은 미등재), PURE_NOVEL (완전 신규)
12 PCPC_Detail str PCPC 분류 상세 설명 "Novel 서열 — 알려진 motif(GHK) 포함하나 전체 서열은 미등재. 신규 INCI 등재 가능."
13 BLAST_Status str NCBI pataa 특허 DB BLAST 결과 patent_full_match (전체 서열 100% 특허 매칭), patent_partial_only (부분만 매칭, 전체 서열은 novel), patent_short_seq (3AA 이하 짧은 서열)
14 Internal_DB_Hits str 내부 펩타이드 DB 교차검증 결과. 23,784개 서열과 대조 "cosmetic_substr(KGH=KGH)" (알려진 motif 포함), "none" (매칭 없음)
FTO Safe CSV: BLOCKED(특허 충돌) + GHK/RGD motif 포함 서열을 제거한 합성 검토용 후보입니다. 원본 CSV에서는 FTO_Status 컬럼으로 각 서열의 특허 상태를 확인할 수 있습니다.

Provenance — 서열 유래 정보 (컬럼 15-23)

# 컬럼명 타입 설명 값 예시
15 Origin_Domain str 서열이 유래한 Desmoplakin(DSP) 단백질 도메인 C-terminal PRD, N-terminal plakin, Central rod, No DSP match
16 Origin_Subdomain str DSP 하위 도메인 PRD-A, PRD-B, PRD-C, Coiled-coil segment 1, inter-domain, N/A
17 Origin_Position str Human DSP P15924 내 position range (1-based) 2033-2040, 1161-1168
18 Match_Length int DSP 원본 서열과의 매칭 길이 (AA) 8, 14
19 Match_Type str 매칭 유형 full (DSP에 동일 서열 존재), partial (부분 매칭), none (매칭 없음)
20 Generation_Phase str 이 서열이 생성된 연구 Phase v7-phase3, v8-phase2, v8-backfill
21 Generation_Method str 서열 생성 방법 DEJ/Recovery Cross-breed, BLOSUM62 Mutation, Conservation-Guided Seed, DSP subsequence scan
22 Is_Natural bool DSP 단백질에 자연적으로 존재하는 서열인지 여부 True (DSP에 exact match), False (mutation/crossbreed로 생성)
23 Cross_Species int 교차 종 수. 서열 생성에 관여한 종의 수 0 (Human only), 1 (Human + 1 lizard species), 2+
Provenance: 모든 펩타이드의 DSP 유래 정보를 추적합니다. Desmoplakin P15924 (Human, 2,871 AA) 서열과의 subsequence matching으로 origin domain과 position을 결정합니다.

Component Scores — 9개 활성 점수 (컬럼 24-32)

# 컬럼명 가중치 설명 범위
24 Collagen 0.22 (22%) Collagen 합성 촉진 활성 예측. FusPBESM2 ML 모델 기반 0~1
25 DSP 0.18 (18%) Desmoplakin interaction score. DSP 단백질과의 상호작용 예측 0~1
26 MMP 0.12 (12%) Matrix Metalloproteinase(MMP) 억제 활성. ECM 분해 방지 0~1
27 AI 0.12 (12%) Anti-inflammatory (항염) 활성 예측 0~1
28 WH 0.10 (10%) Wound Healing similarity. 상처 치유 관련 펩타이드와의 유사도 0~1
29 AOX 0.08 (8%) Antioxidant (항산화) 활성 예측 0~1
30 CPP 0.08 (8%) Cell Penetrating Peptide score. 세포 침투 능력 (R, K 양전하 잔기 비율 기반) 0~1
31 Stab 0.06 (6%) Stability score. 펩타이드 안정성 (instability index 기반) 0~1
32 Skin 0.04 (4%) Skin penetration score. 피부 침투 예측 (분자량, GRAVY, 전하 기반) 0~1
Score = Σ(Component × Weight): 9개 component의 가중합. Activity Triad (Collagen + DSP + MMP) = 52%로 DEJ 구조 회복 관련 활성이 스코어의 과반을 차지합니다. 모든 component score는 0~1 범위의 정규화된 값입니다.
