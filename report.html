연구 보고서 v3 피드백 요청서 v2

Abstract

본 연구는 Podarcis muralis 및 근연종 3종(Anolis carolinensis, Podarcis lilfordi)의 Desmoplakin homolog 10종과 Human Desmoplakin(P15924)에서 화장품 활성 펩타이드를 AI 기반으로 탐색하였다.

총 6개 연구 버전(v1~v6), 23개 세부 Phase를 거쳐:

  • **8가지 상호보완적 추출/분석 방법론** 적용
  • v1~v4에서 독립적으로 **~97,000개 펩타이드** 마이닝 → **v5에서 전 버전 합산 후 서열 기준 중복 제거 → 5,423개 고유 서열** 확보
  • **2개 자체 ML 활성 예측 모델** 개발:
  • Collagen Activity: FusPBESM2 v4 (AUC-ROC 0.9555, 외부 벤치마크 32개) — v1~v6 전 과정 적용
  • CPP: FusPBESM2 CPP v3 (val AUC 0.9606, test 0.9349) — v6에서 Rule v2와 앙상블(35/65) 적용
  • 스코어링 체계 4차 개선 (Unified → Longevity → Viability → Viability+CPP_v2)
  • ESMFold 기반 구조 검증 gate 적용

최종(v6) 기준 최고 점수 펩타이드는 **VQLK**(Viability **0.788**, C-grade)이며, Collagen Activity 0.965와 ML CPP 0.999를 동시 달성하였다. 이전 1위 ARIK(0.735, C-grade)는 ML CPP 모델이 보수적으로 재평가하여 하향 조정되었다. 본 보고서는 in silico 평가 결과를 종합하여 합성 우선순위 펩타이드와 향후 in vitro 검증 방향을 제안한다.

1. 연구 배경 및 목적

1.1 Podarcis muralis Desmoplakin과 재생 생물학

Podarcis muralis는 꼬리 autotomy 후 완전한 재생을 보이는 파충류로, 이 과정에서 Desmosome remodeling이 핵심 역할을 한다 (Alibardi, 2020 [11]). Desmoplakin은 desmosome의 핵심 구성 단백질로, 중간섬유(intermediate filament)를 세포막에 고정하며, 피부 상피 조직의 구조적 완전성을 유지한다. 본 연구는 이 종 및 근연종의 Desmoplakin에서 화장품 활성 펩타이드를 발굴하여, 인간 피부 재생에 활용하는 것을 목표로 한다.

1.2 연구 목표

  1. 3종 파충류 + Human Desmoplakin 10종에서 다중 활성 펩타이드 탐색
  2. Collagen activity + CPP + stability + skin penetration + LL37 similarity + conservation + anti-aging similarity 다중 최적화
  3. ML 기반 활성 예측 모델 개발 및 적용:
  • **Collagen Activity ML** (v1~v6 적용): FusPBESM2 Collagen Head v4 (AUC-ROC 0.9555, 외부 벤치마크 32개)
  • **CPP ML** (v6 적용): FusPBESM2 CPP Head v3 (val AUC 0.9606) + Rule v2 6-component Ensemble
  1. A-grade (≥ 0.90) 펩타이드 확보 및 합성 우선순위 도출
  2. ESMFold pLDDT 기반 구조 검증 gate 적용
  3. 마케팅 포지셔닝별 추출 방법론 분류 및 전략 제안

**Conservation Score 정의**: 4종(Human, *A. carolinensis*, *P. lilfordi*, *P. muralis*)의 Desmoplakin 서열을 다중 정렬(MSA) 후, 도메인 내 각 아미노산 위치에서 **종간 일치율의 평균**을 산출한 값(0~1).

- **1.000** = 4종 모든 위치에서 아미노산이 **완전히 동일** (예: Linker B-C, 16 aa — 3억 년+ 종분화에도 변화 없음)
- **0.807** = 약 81%의 위치에서 동일 또는 유사 (예: C-terminal PRD, 1,115 aa)
- **0.610** = 약 61%만 보존 — 상대적으로 진화적 변동이 큰 영역 (예: Central Rod)
- **해석**: 높은 보존도 = 기능적으로 필수적인 서열. 보존된 영역에서 추출된 펩타이드는 종간 기능이 유지될 가능성이 높으며, 생물학적 활성의 진화적 근거를 제공함.

**연구 시리즈 간 피드백**: v1 결과 → v2 확장 전략 → v2 A-grade 달성 → v3 Human DSP 집중 → v4 도메인/Longevity 시도 → Longevity 천장 발견 → v5 Viability 재설계 + ML 리뷰 반영 → v6 CPP ML 적용. **반복적 개선(iterative refinement)** 구조.

**펩타이드 카운트 해설**:
- **v1~v4 (Discovery)**: 각 버전이 독립적으로 마이닝. 동일 서열이 중복 존재 가능.

- **v6 (CPP Re-score)**: v5와 동일 5,423개, CPP component만 Ensemble로 재평가.
- **v4→v5 감소**: v4 단독 65,938개이나, v5에서 v4 Phase 2 출력(~5,400개)만 로드. 전 버전 합산 → 중복 제거 후 5,423개.

5. 핵심 발견 및 인사이트 (Discussion)

5.1 VQLK의 부상 — Amphipathic 패턴의 중요성

VQLK(V-Q-L-K)는 ML CPP에서 **0.999**라는 극도로 높은 점수를 받았다:

  • **V(Val)**: 소수성, **Q(Gln)**: 극성, **L(Leu)**: 소수성, **K(Lys)**: 양전하
  • 소수성-극성-소수성-양전하의 교번 패턴 → 전형적인 amphipathic arrangement
  • 동시에 Collagen Activity 0.965(Top 5 수준)

이 결과는 CPP 활성에서 단순 양전하(R/K) 비율보다 **amphipathic 배열**이 더 중요한 결정 요인임을 시사한다 (Guidotti et al., 2017 [6]).

5.2 ARIK의 재평가 — 4-mer OOD 문제

ARIK(A-R-I-K)는 v1~v5에서 최고 점수를 유지했으나, v6에서 CPP가 0.960 → 0.575로 급락:

  • ML CPP v3 학습 데이터 대부분이 5+ AA → 4-mer는 OOD 영역
  • Rule v2도 4-mer length penalty 적용
  • 그러나 Collagen Activity **0.990**은 여전히 전체 1위 → 생물활성 가치 유효

4-mer CPP의 실험적 검증이 필요하며, FITC 표지 세포 투과 실험으로 in silico 예측을 확인해야 한다.

5.3 ML이 non-CPP로 판정한 펩타이드들 (IKKV, RGLV, RNLV)

이 3개 펩타이드는 ML CPP v3에서 0.004~0.017 점수를 받았다:

  • **IKKV**(I-K-K-V): K 2개가 연속(KK) → ML이 CPP 기능 구조로 미인식
  • **RGLV**(R-G-L-V): R 1개, G는 중성 → amphipathic 패턴 부족
  • **RNLV**(R-N-L-V): R 1개, N은 극성이나 CPP motif와 무관

Rule v1에서는 R/K 비율만으로 일괄 0.879를 부여했으나, ML은 실제 CPP 데이터베이스(5,349 seq)에서 이런 서열이 CPP로 분류되지 않음을 학습하였다.

5.4 4AA vs 5-7AA 길이 트레이드오프

v6의 ML 결과는 최적 길이를 재고하게 한다:

  • **4AA 장점**: MW < 500 → Skin penetration 보장 (Rule of 500 [9]), 안정성 높음, 합성 용이
  • **4AA 단점**: CPP ML 학습 데이터 OOD, amphipathic helix 형성 어려움
  • **5-7AA 재부상**: GVKGKK(6AA, 0.724), VPRVR(5AA, 0.718), LLPLKEK(7AA, 0.707) 등이 v6 상위권에 진입

5.5 Human DSP의 A-grade 불가 분석

v3에서 22,703개 Human DSP 유래 펩타이드를 탐색했으나 A-grade 0개:

  • Human DSP PRD에서 R/K 밀집 4-mer가 적음 → CPP 제한
  • Conservation은 Human origin에서도 0.06 수준
  • 단, 규제 및 마케팅 측면에서 Human-derived 펩타이드(GVKGKK 등)의 가치는 높음

  2. **5-7mer 확장 전수조사**: VQLK/LVPR 기반 constrained search
  3. **CPP ML 모델 개선**: 4-mer 전용 fine-tuning 또는 4-mer enriched dataset 구축
  4. **MD/Docking**: 상위 후보 분자동역학 시뮬레이션 (Collagen receptor binding)

  6. **특허 검토**: VQLK/ARIK의 화장품 용도 특허 출원

  • MW, GRAVY, charge, length 복합 (Rule of 500 [9], Lipinski [3])
